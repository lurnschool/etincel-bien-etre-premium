----------------------------------------------------------------
CRITÈRES DE SÉLECTION (brief chef de projet)
----------------------------------------------------------------

- ambiance méditation / retraite / refuge / soin
- douce, organique
- sans voix
- sans percussion agressive
- pas de "spa cheap" ni dramatique
- durée minimale 3 minutes
- boucle possible
- format MP3 local (pas de streaming externe)

----------------------------------------------------------------
INSTALLATION
----------------------------------------------------------------

1. Télécharger un MP3 conforme depuis l'une des sources ci-dessus.
2. Renommer le fichier en : ambiance-etincel.mp3
3. Le placer ici : /public/audio/ambiance-etincel.mp3
4. Renseigner ce fichier (titre, auteur, source, licence, date).
5. Dans src/components/layout/AmbianceToggle.tsx, mettre :
      const AUDIO_SRC = "/audio/ambiance-etincel.mp3";
6. Si la licence requiert une attribution, l'ajouter dans le
   footer du site (src/components/layout/Footer.tsx).

Le composant AmbianceToggle bascule automatiquement de "Ambiance
bientôt disponible" à "Activer l'ambiance" dès que AUDIO_SRC est
renseigné et qu'un fichier répond à l'URL.
